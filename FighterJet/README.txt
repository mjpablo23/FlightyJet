README

Instructions:
Tap the screen repeatedly to fly through the barriers. 

Main files are:
1) FighterJetViewController.swift:  
This is the main file.
This files contains the animations of various objects, and draws the jet, barriers, and ufo.

2) JetBehavior.swift: 
This file describes the UI dynamic behaviors for the various items like the jet, bullet, and ufo.  
Behaviors include gravity, push behaviors at different magnitudes and angles.

3) StartScreenViewController.swift
This file is for the screen that pops up in between games.  
buttons to restart, continue, give feedback by text, rate the app, credits alert

4) BezierPathsView.swift
This file draws the bezier paths used in the background.

5) DeviceVersion.swift
File to detect iphone version for different screen sizes
